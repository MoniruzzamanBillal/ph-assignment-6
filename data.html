Q1: Discuss the scope of var, let, and const.

Answer:

In JavaScript, "scope" refers to the context in which variables are declared and can be accessed.

The scope of var, let, and const :

1. Var: The scope of a variable declared with var is function scope. which means they are visible throughout the entire function where they are defined

2. Let: The scope of a variable declared with let is block scope. which means they are limited to the code block (like a function or a loop) where they are defined. This is more controlled and helps prevent issues caused by variable leakage.

3. Const: The scope of a variable declared with const is also block scope. We use const when we want to create a variable that shouldn't change its value.

Q2: Tell us the use cases of null and undefined.

Answer:

Null is used to explicitly represent the absence of a value.

Undefined is used to represent the absence of a value when a variable has not been assigned a value.

use cases of null:

1.Explicit Empty Value: Use null to indicate that a variable or property intentionally has no value.

2.Placeholder: Use null as a temporary placeholder when you plan to assign a value to a variable or property later.

use cases of undefined:

1.Default Value: Variables that are declared but not assigned a value automatically have the value undefined.

2.Function Parameters: When a function is called with missing arguments, those parameters are set to undefined.

3.Missing Object Properties: When trying to access a property that doesn't exist on an object, it returns undefined.

Q3:What do you mean by REST API?

Answer:

REST API stands for Representational State Transfer Application Programming Interface.

REST API is a way for different computer programs to communicate and share data over the internet. It uses a set of rules and conventions, like using URLs to represent resources (like web pages) and standard HTTP methods (like GET, POST, DELETE) to perform actions on these resources. Think of it as a universal language that allows computers to talk to each other and exchange information.
